    Holo Saltmine API Documentation

    Welcome to the Holo Saltmine API! You can use our API to access Holo Saltmine API endpoints.

    • Every API in this service will always return 32 hex encoded bytes with mime type text/plain.
    • POST requests will require a "Content-Type" header of "application/x-www-form-urlencoded".
    • Response headers will include an "Access-Control-Allow-Origin" header of "*" for Cross-Origin Resource Sharing (CORS).
    At present, we have language bindings for command line / shell scripts! (You can view code examples in the dark area to the right.)

    Entropy

    Any GET request to this service will return 32 bytes Web Crypto random bytes.

    (We provide this service because not all devices can produce cryptographically secure entropy).

    • Method
      • GET
    • Request Parameters
      • none/ignored
    • Responses
      • 32 bytes Web Crypto random bytes
      • error

    HTTP Request

    GET https://saltmine.holohost.net

    URL Parameters

    URL parameters are ignored for this request.

    Query Parameters

    Query parameters are ignored for this request.

    curl  -X GET "https://saltmine.holohost.net"

    The above command returns 32 hex encoded bytes with mime type text/plain

    Errors

    Errors are returned when there is a problem with the request received.

    The Holo Saltmine API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Your credentials are invalid.
    403 Forbidden -- Your credentials are valid but you do not have permission to access the requested resource.
    404 Not Found -- The specified kitten could not be found.
    415 Unsupported Media Type -- Returned when Content-Type header is missing or incorrect.
    418 I'm a teapot.
    500 Internal Server Error -- We had a problem with our server. Try again later.
